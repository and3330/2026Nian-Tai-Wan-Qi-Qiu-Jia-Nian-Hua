## 區塊二：Meta 廣告後台設定教學

從零開始的保姆級步驟，新手也能輕鬆上手。

---

### 步驟 1：建立 Meta 商業帳號（Business Manager）

1. 前往 business.facebook.com 並登入你的 Facebook 帳號
2. 點選「建立帳號」，輸入商業名稱（例如：2026臺灣氣球嘉年華）
3. 填入你的姓名和電子信箱，按下「提交」
4. 前往信箱驗證電子郵件，完成帳號建立

---

### 步驟 2：建立廣告帳號

1. 進入 Business Manager 後，點選左側「商業設定」
2. 選擇「帳號」→「廣告帳號」→「新增」
3. 選擇「建立新的廣告帳號」
4. 輸入廣告帳號名稱、選擇時區（GMT+8 台北）和幣別（TWD 新台幣）
5. 點選「建立」完成設定

---

### 步驟 3：設定付款方式

1. 進入廣告帳號設定頁面，點選「付款設定」
2. 點選「新增付款方式」
3. 可選擇信用卡/簽帳金融卡（建議使用），或 PayPal
4. 輸入信用卡資訊並儲存
5. 設定帳單上限（建議新手先設 NT$1,000 - NT$3,000 以控制預算）

---

### 步驟 4：建立新廣告活動（Campaign）

1. 前往 Meta 廣告管理員（adsmanager.facebook.com）
2. 點選綠色的「+ 建立」按鈕
3. 選擇行銷目標：建議選擇「流量」（導引人潮到報名頁）或「轉換次數」（追蹤實際報名人數）
4. 為活動命名，例如「氣球嘉年華_7月_導流」
5. 設定廣告活動預算：建議選擇「每日預算」，新手建議從 NT$200 - NT$500/天 開始測試
6. 點選「下一步」繼續設定廣告組合

> **預算建議**
> - 小額測試方案：先用 NT$200/天 跑 3 天觀察成效，若點擊率 > 2% 再加碼到 NT$500/天
> - 建議活動前 2-4 週開始投放，預留足夠時間累積曝光
> - 總預算建議：NT$3,000 - NT$10,000（依需求調整）

---

### 步驟 5：設定廣告組合（Ad Set）

1. 為廣告組合命名，例如「北部家長_親子興趣」
2. 設定受眾條件：

**受眾設定建議**

| 設定項目 | 建議值 |
|---------|-------|
| 地區 | 台灣 → 台北市、新北市、基隆市、桃園市（北部地區） |
| 年齡 | 25 - 45 歲 |
| 性別 | 不限（或可針對女性投放，通常媽媽族群互動率更高） |
| 詳細目標設定（興趣） | 親子活動、家庭旅遊、兒童教育、手作DIY、親子餐廳、育兒、媽媽社團、兒童遊樂、暑假活動 |

3. 設定排程：建議「開始日期」設在活動前 2-4 週，「結束日期」設在 7/24（活動前一天）
4. 版位選擇：建議選「自動版位（Advantage+ 版位）」讓系統自動優化，或手動選擇 Facebook 動態消息 + Instagram 動態消息 + Instagram 限時動態
5. 點選「下一步」進入廣告設定

---

### 步驟 6：建立廣告（Ad）

1. 為廣告命名，例如「溫馨家庭風_文案A」
2. 選擇你的 Facebook 粉絲專頁和 Instagram 帳號
3. 廣告格式：建議選擇「單一圖片或影片」（最簡單好操作）
4. 上傳素材：上傳你準備好的圖片或影片（建議尺寸：1080×1080 正方形 或 1080×1920 限時動態）
5. 填入廣告文案：將上方提供的文案複製貼上到對應欄位（主要文字、標題、說明）
6. 設定行動呼籲按鈕：選擇「瞭解詳情」或「註冊」
7. 設定網站網址：填入報名頁網址 https://www.balloon-carnival.tw/carnival（記得加上 UTM 參數，見區塊三）
8. 點選「預覽」確認廣告在各版位的顯示效果
9. 確認無誤後，點選「發佈」送出廣告審查（通常 24 小時內完成審查）

---

### 步驟 7：追蹤廣告成效（基礎指標）

1. 發佈後回到廣告管理員首頁，即可看到廣告成效數據
2. 重要指標說明：

| 指標 | 說明 |
|-----|------|
| 曝光次數（Impressions） | 廣告被顯示的總次數 |
| 觸及人數（Reach） | 看到廣告的不重複人數 |
| 連結點擊次數（Link Clicks） | 點擊廣告連結前往報名頁的次數，這是最重要的指標 |
| 點擊率（CTR） | 點擊次數 ÷ 曝光次數。好的 CTR 通常 > 1%，優秀的 > 2% |
| 每次點擊成本（CPC） | 每一次連結點擊花費多少錢。台灣市場通常 NT$3 - NT$15 |
| 花費金額（Amount Spent） | 累計花費的廣告費用 |

3. 建議每天檢查一次成效，若某組廣告 CTR 低於 1%，可考慮更換素材或文案
4. 表現好的廣告可以增加預算，表現差的可以暫停

## 區塊三：報名流程引導

從廣告點擊到完成報名的完整使用者旅程。

---

### 使用者旅程

1. **看到廣告** — 使用者在 Facebook 或 Instagram 動態中看到氣球嘉年華廣告
2. **點擊廣告** — 被文案與圖片吸引，點擊「瞭解詳情」或「立即搶票」按鈕
3. **進入報名頁** — 跳轉到 balloon-carnival.tw/carnival 的購票區域
4. **選擇票種** — 選擇單日票（NT$200）或兩日套票（NT$300），查看各日剩餘名額
5. **填寫資料** — 填入姓名、聯絡電話、購票張數等基本資訊
6. **確認購票** — 送出表單，系統即時扣減名額並顯示購票成功畫面

### UTM 參數設定建議

UTM 參數可以幫你追蹤每則廣告帶來多少流量。在 Meta 廣告後台的「網站網址」欄位填入以下完整網址：

**UTM 參數說明**

| 參數 | 用途 |
|-----|------|
| `utm_source` | 流量來源（facebook 或 instagram） |
| `utm_medium` | 媒介類型（paid = 付費廣告） |
| `utm_campaign` | 活動名稱（用於辨識這次的投放活動） |
| `utm_content` | 廣告內容（用於辨識不同風格的廣告文案） |

**完整網址範例**

**Facebook 廣告 - 溫馨家庭風：**
```
https://www.balloon-carnival.tw/carnival?utm_source=facebook&utm_medium=paid&utm_campaign=balloon_carnival_2026&utm_content=family_warm
```

**Facebook 廣告 - 限量緊迫風：**
```
https://www.balloon-carnival.tw/carnival?utm_source=facebook&utm_medium=paid&utm_campaign=balloon_carnival_2026&utm_content=scarcity_urgent
```

**Instagram 廣告 - 活動亮點風：**
```
https://www.balloon-carnival.tw/carnival?utm_source=instagram&utm_medium=paid&utm_campaign=balloon_carnival_2026&utm_content=highlights_fun
```
